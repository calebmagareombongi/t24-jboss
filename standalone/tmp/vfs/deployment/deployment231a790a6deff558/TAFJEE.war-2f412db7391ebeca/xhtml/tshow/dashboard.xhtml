You can get routine information by submiting your routine name in the field below.

To introspect a ComponentMethod pass the '-m' flag. Example : -m EB.SystemTables.GetAlternateKeyDets

To introspect a L3 java class, '-j' flag. Example : -j com.thebank.TheL3JavaClass

To introspect a Record class, '-record' flag. Example : -record AaProperty
