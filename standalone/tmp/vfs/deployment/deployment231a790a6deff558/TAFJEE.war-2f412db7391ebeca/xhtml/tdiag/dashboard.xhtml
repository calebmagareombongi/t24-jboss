tDiag servlet

You can get information about your TAFJ environment. Please refer to your TAFJ properties file and to TAFJ documentation for more information about these properties.

Properties values being displayed in red are null or undefined in the properties file and have been defaulted to their default value.

Values being displayed in red and not related to the properties file are pointing some inconsistency in the configuration and need to be reviewed.
